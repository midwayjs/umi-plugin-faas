# umi-plugin-fas

Umi plugin for faas.

## Install

```bash
yarn add umi-plugin-faas
# OR npm install umi-plugin-faas
```

## Usage

```
export default {
  faas: {
    
  },
};
```

## Options

See [midway faas dev pack](https://github.com/midwayjs/midway-faas/tree/master/packages/faas-dev-pack) for `options` object to pass to the plugin.
